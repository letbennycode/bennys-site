# Story 1.5: Design System & Aesthetic Foundation

## Description
Establish the complete design system and visual identity inspired by Tom Sachs' minimal, industrial aesthetic. This story creates the foundation for all visual elements, positioning the site as a "creative and culture engineer firm" with a utilitarian, functional design language.

## Acceptance Criteria
- [ ] Typography system defined (fonts, sizes, weights, styles)
- [ ] Color palette established (monochromatic/limited, industrial)
- [ ] Spacing system configured in Tailwind
- [ ] Design tokens documented
- [ ] Global styles reflect Tom Sachs aesthetic
- [ ] Component design patterns established
- [ ] Brand positioning visually communicated
- [ ] Design system is consistent across all elements

## Technical Details

### Typography System

**Font Selection:**
- Primary: Bold, industrial sans-serif (consider: Inter, Helvetica, or similar)
- Uppercase for navigation and headings
- Minimal font weights (400, 700)
- Large, impactful sizes for hero/headings
- Clean, readable body text

**Typography Scale:**
```typescript
// Tailwind typography configuration
fontSize: {
  'xs': ['0.75rem', { lineHeight: '1rem' }],
  'sm': ['0.875rem', { lineHeight: '1.25rem' }],
  'base': ['1rem', { lineHeight: '1.5rem' }],
  'lg': ['1.125rem', { lineHeight: '1.75rem' }],
  'xl': ['1.25rem', { lineHeight: '1.75rem' }],
  '2xl': ['1.5rem', { lineHeight: '2rem' }],
  '3xl': ['2rem', { lineHeight: '2.5rem' }],
  '4xl': ['2.5rem', { lineHeight: '3rem' }],
  '5xl': ['3rem', { lineHeight: '1' }],
  '6xl': ['3.75rem', { lineHeight: '1' }],
  '7xl': ['4.5rem', { lineHeight: '1' }],
  '8xl': ['6rem', { lineHeight: '1' }],
  '9xl': ['8rem', { lineHeight: '1' }],

```

**Typography Usage:**
- Navigation: Uppercase, bold, medium size
- Headings: Uppercase or title case, bold, large sizes
- Body: Sentence case, regular weight, readable size
- Labels: Uppercase, small, bold

### Color Palette

**Tom Sachs-Inspired Palette:**
- Primary: Black (#000000) - text, borders, emphasis
- Background: White (#FFFFFF) - primary background
- Secondary: Gray scale (various shades for depth)
- Accent: Minimal, if any (consider single accent color for links/hover)

### Spacing System

**Tailwind Spacing:**
- Use Tailwind's default spacing scale
- Generous whitespace (Tom Sachs aesthetic)
- Consistent spacing between elements
- Large margins for hero sections
- Tight spacing for navigation items

### Design Principles

1. **Minimalism**
   - Remove all unnecessary elements
   - Focus on content and function
   - Clean, uncluttered layouts

2. **Utilitarian**
   - Functional over decorative
   - Clear hierarchy
   - Direct communication

3. **Industrial**
   - Bold, strong typography
   - Geometric shapes
   - High contrast
   - Monochromatic palette

4. **Culture Engineering**
   - Professional yet creative
   - Communicates expertise and vision
   - Builds trust through clarity

### Component Design Patterns

**Navigation:**
- Horizontal layout
- Uppercase, bold text
- Minimal spacing between items
- Simple hover states (underline or color change)

**Buttons/Links:**
- Minimal styling
- Uppercase text
- Simple hover effects
- No rounded corners (sharp, geometric)

**Cards/Containers:**
- Clean borders (thin, black)
- Generous padding
- No shadows or gradients
- Simple, functional layout

**Images:**
- Full-width or large sections
- High contrast
- No rounded corners
- Minimal captions

### Files to Create/Modify

- `tailwind.config.ts` - Extended with custom colors, typography, spacing
- `app/globals.css` - Global styles, typography classes, design tokens
- `lib/design-tokens.ts` - TypeScript design tokens (optional, for consistency)
- `docs/design-system.md` - Design system documentation (optional)

### Global Styles

```css
/* Example global styles */
@layer base {
  body {
    @apply font-sans antialiased;
    @apply text-black bg-white;
  }
  
  h1, h2, h3, h4, h5, h6 {
    @apply font-bold uppercase tracking-tight;
  }
  
  a {
    @apply uppercase font-bold;
    @apply hover:underline;
  }
}
```

### Brand Positioning

Visual elements should communicate:
- **Creative**: Bold typography, artistic layouts, visual impact
- **Culture Engineer**: Professional, structured, thoughtful
- **Firm**: Established, trustworthy, serious
- **Future Products**: Space for growth, scalable design system
